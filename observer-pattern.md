# 观察者模式

| 名称                  | 解释        |
| --------------------- | ----------- |
| subject               | 主题        |
| observer              | 观察者      |
| observable            | 可观察的    |
| notify                | 通知        |
| update                | 更新        |
| listener              | 监听器      |
| register / unregister | 注册 / 注销 |
| publish / subscribe   | 发布 / 订阅 |
| send / receive        | 发送 / 接收 |
